A modular approach

A modular design approach was adopted, focusing on creating a flexible and user-centric experience. My process began with an extensive benchmarking and analysis phase. Although there were few direct competitors in the NGO space, several organizations were examined to gather insights. Additionally, NGOs from other sectors were explored that engage with corporates for CSR activities to identify best practices and trends.

Following this, I conducted a collaborative brainstorming session with the client, complemented by a card sorting activity. This helped us define the optimal information architecture and user flow. Utilizing these insights, I developed an initial box diagram to outline the website structure. This diagram was continuously refined based on client feedback, eventually evolving into the final wireframes.

With the foundational layout established, I proceeded to create detailed sketches for each module of the website. This modular design approach allowed us to focus on each section individually, ensuring that each module effectively addressed user needs and client goals.
